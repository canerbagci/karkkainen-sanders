Karkkainen-Sanders suffix array generation 

Made in python

info:
-can read plain text and FASTA format

Code is run from the terminal:

python code/python/projekt_v4.py tests/test1.txt out.txt
- where tests/test1.txt is the input file, out.txt is the output file,and code/python/projekt_v4.py  is the python script 

script terminates with an error
 - if the number input string is in the incorrect format
 - if the number of command line parameters isn't correct

Please refer to the README.txt in the main directory for more read to c/p usage examples
