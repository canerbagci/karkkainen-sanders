Program prima dva ulazna parametra:
1. putanja do ulazne datoteke u kojoj se nalazi string cije sufiksno polje zelimo saznati (input.txt)
2. putanja do izlazne datoteke u koju se zapisuje dobiveno sufiksno polje (output.txt)

Datoteka input.txt koja je prilozena je ulazna datoteka u kojoj se nalazi FASTA sekvenca. 
Datoteka output.txt koja je prilozena je prazna izlazna datoteka u koju ce se zapisati sufiksno polje.
